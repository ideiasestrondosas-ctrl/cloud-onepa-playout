Cloud Onepa Playout PRO

1. Introdução

O Cloud Onepa Playout é uma solução profissional de emissão televisiva baseada na web, projetada para automação de canais 24/7 com suporte a HLS, RTMP e SRT.

Dica: Para gerar um PDF deste manual, pressione Ctrl + P (Windows) ou Cmd + P (Mac) no seu browser e escolha "Guardar como PDF".

2. Dashboard de Emissão

O centro de controlo do seu canal. Aqui pode monitorizar o sinal de saída e gerir os clips em exibição.

Funcionalidades de Phase 31:

  • Monitor de Áudio Standardizado: Visualização em tempo real com alvo de -23 LUFS (Norma EBU R128).
  • Botão Skip: Pula imediatamente para o próximo clip agendado, sem interrupções pretas.
  • Smart VLC Launcher: Botão "Open VLC" que deteta o seu sistema operativo e abre o player localmente com as configurações corretas.

3. Configurações e Overlay

Personalize a identidade visual do seu canal em tempo real.

Novidade: As alterações de Opacidade e Escala do logotipo agora são aplicadas instantaneamente ao stream. O sistema reinicia o processo de vídeo de forma seamless para refletir as definições.

Parâmetros Recomendados:

Parâmetro Valor Recomendado Descrição
Resolução 1920x1080 Full HD standard.
Bitrate Vídeo 4000k Equilíbrio entre qualidade e largura de banda.
Normalização -23 LUFS Ativada automaticamente em todos os clips.

4. Resolução de Problemas

O sistema inclui um Assistente de Diagnóstico que verifica:

  1. Se o motor de playout está ligado.
  2. Se existe uma playlist agendada para o horário atual.
  3. Se os ficheiros físicos de vídeo existem no servidor Cloud.
  4. A integridade das permissões de ficheiros.
